Comment utiliser OPROW ?

Pour utiliser l'application, vous devez rentrer les administrations dans lesquelles vous vous rendez fréquement. Pour cela vous aller dans "Ajouter une administration" et vous pouvez utiliser la barre de recherche.

Une fois que vous avez sélectionné vos administrations en favoris, elles apparaitront sur la page d'accueil. En cliquant sur l'une d'entre elles vous aurez accès à des informations sur l'affluence en fonction du jour et de l'heure. Vous pouvez donc facilement choisir la période à laquelle vous vous rendrez dans l'administration.
